# ESTET DENT — лендинг частной стоматологии

Премиальный одностраничник для стоматологической клиники: тёмная тема с золотыми акцентами, шрифтовая пара Cormorant Garamond + Manrope.

**Демо:** https://trip-prog.github.io/estet-dent-landing/

## Что внутри

- Hero с формой онлайн-записи (маска телефона, валидация, выбор даты и времени)
- Интерактивные кейсы «до/после» с перетаскиваемым слайдером сравнения
- Анимированные счётчики статистики
- Услуги, врачи, цены, отзывы, CTA-блок
- Reveal-анимации при скролле
- Полностью адаптивная вёрстка (десктоп / планшет / мобильный)

## Стек

Чистые HTML + CSS + JS, без сборщиков и зависимостей. Иллюстрации «до/после» — инлайновые SVG (в боевой версии заменяются на фото пациентов).

## Запуск локально

```
python -m http.server 8080
```

и открыть http://localhost:8080
